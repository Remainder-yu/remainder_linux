# 内存之路

## 概述

该目录的背景是为了学习linux内存管理模块，进行学习记录。该模块与进程管理为linux重点以及复杂模块，也是技术基础。我想利用自学路程进行记录整理，从系统层面整体认知，再到各个子模块以及功能进行攻克，通过个人理解绘制流程图、系统框图等示意图。然后结合代码调用流程以及数据结构关系输出相关文档，而脱离要么只讲概念，要么只讲代码。通过个人的积累，输出达到博客级文档。
通过整个模块的积累，然后学习调试分析手段；
针对主要场景，梳理出流程化分析思路；整理调试工具及使用；输出分析总结文档；
通过整个模块的整理学习，思考调测调度等尝试发表论文或专利；这是针对该模块掌握最好的输出，如果可能情况，可以内存优化领域深耕，但是目前职业规划并未有此计划，但是内存管理是重要模块，所以在此停留。
